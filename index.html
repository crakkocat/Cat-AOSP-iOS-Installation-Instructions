📘 README Documentation

Cat-AOSP is a custom fork of Dolphin Emulator focused on stability, performance tuning, cross-platform builds, and iOS support.

📦 Available Platforms

  • Windows (x64 / ARM64)
  • macOS
  • Linux
  • Android (APK)
  • iOS (IPA – sideload required)

🍎 iOS Installation Guide

⚠ The IPA must be sideloaded. It cannot be installed directly.

🪟 Windows

Requirements

  • Windows 10 or later
  • iTunes (Non-Microsoft Store version):
    https://support.apple.com/en-us/106372
  • iCloud (Non-Microsoft Store version):
    Download iCloud

Do NOT use Microsoft Store versions.

Sideloading Tools

  • 3uTools
  • AltServer
  • Sideloadly (Recommended – supports auto-refresh)

Steps

  1. Install iTunes and iCloud.
  2. Install a sideloading tool.
  3. Connect your iOS device.
  4. Sign the Cat-AOSP IPA.
  5. Install to device.
  6. Go to Settings → General → VPN & Device Management.
  7. Trust your developer profile.
  8. Launch Cat-AOSP.

🍎 macOS

Requirements

  • macOS 10.14 or lower (iTunes required)
  • iCloud (pre-installed)

Tools

  • 3uTools
  • AltServer
  • Sideloadly (Recommended)

Steps

  1. Install a sideloading tool.
  2. Connect your device.
  3. Sideload the Cat-AOSP IPA.
  4. Trust the developer profile.
  5. Launch the app.

🐧 Linux

Requirements

  • Any Linux distribution
  • Legacy-iOS-Kit

Installation Steps

cd <LegacyiOSKit directory>
bash restore.sh

Or:

./restore.sh
  1. Enter your password if prompted.
  2. Select App Management.
  3. Choose the Cat-AOSP IPA.
  4. Sign the app.
  5. Launch it on your device.

⚠ Important Notes

  • Free Apple developer accounts expire every 7 days.
  • Re-signing is required unless using a paid Apple Developer account.
  • Sideloadly auto-refresh is recommended.
